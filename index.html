How it works

01

Upload locally

Photos are loaded directly into your browser memory using the File API. Nothing is sent anywhere.

02

Choose your clean

Strip everything, randomize with fake data, set custom values, or just remove GPS. Your call.

03

Re-drawn, not edited

Each image is redrawn onto a canvas element. The output is a brand-new image file with zero metadata.

04

Download & done

Get all cleaned photos in a single ZIP file. Fast, private, and 100% yours.
